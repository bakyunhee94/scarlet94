기존 코드의 유지보수성과 재사용성을 높이기 위해 전반적인 리팩토링을 수행하였습니다.

**주요 업무:**

-   사내 TMS(운송관리시스템) 솔루션의 화면 UI 설계 및 주요 기능 흐름 정의
-   프론트엔드와 백엔드 연동을 고려한 API 설계 및 구현
-   코드의 유지보수성과 확장성을 고려한 전반적인 리팩토링 수행
